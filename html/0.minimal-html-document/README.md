# Что и чем называется

* Это тег - `<html>`
* Это закрывающий тег - `</html>`
* Это элемент - `<html lang="ru">А этот текст - содержимое элемента</html>`
* Это атрибут - `lang`
* Это значение атрибута - `"ru"`

# \<!DOCTYLE html>

В начале любого HTML5 документа должно быть `<!DOCTYPE html>`, доктайп говорит браузеру, что ниже начинается HTML5 документ.
Сейчас HTML5 современный стандарт, для его указания в доктайпе пишется просто `html`.

# \<html>

`<html>` - собственно начало HTML5 документа, корневой элемент (root - корень). Он имеет атрибут `lang`, значение которого - `ru`. - Это объясняет браузеру, что основной язык сайта - русский, а любые другие языки - иностранные и их можно, например, предложить перевести.

# \<head>

Служебная часть страницы. Никак не отображается визуально. Здесь лежат любые метаданные, помогающие характеризовать и описать сайт. - Это важно. Браузер хочет знать: "Как называть открытую вкладку?", поисковик Google, показывая результаты поиска, хочет понять: "Где заголовок и где описание сайта?".

# \<meta>

Единица метаданных. Например `<meta charset="utf-8">` говорит браузеру, что весь этот документ написан в современной кодировке UTF-8. А подсказать описание сайта Google для поисковой выдачи, или Telegram для карточки сайта, когда кто-то вставляет ссылку на сайт в чат, можно так: `<meta name="description" content="Описание сайта">`.

# \<body>

Здесь содержится все, что видно пользователю.
